NAME

sjgllghr -- a github user

SYNOPSIS

sjgllghr [-fv] [-A num]

DESCRIPTION

The sjgllghr user is a person who writes code and stuff. They have a Bachelor's of Science in Computer Science from the University of Washington.

The following options are available:

-A num, --arg-again=num

Gonna write a bunch of text and make sure the formatting still works. Gonna write a bunch of text and make sure the formatting still works. Gonna write a bunch of text and make sure the formatting still works.

-f, --forced

Have sjgllghr admit that maybe this metaphor is a bit forced. (But it was fun to write the code.)

-v, --verbose

Have sjgllghr give more verbose output.
